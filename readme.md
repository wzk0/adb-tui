# adb-TUI

> 一个彩色的adb的终端界面, 用于操作adb批量卸载软件或查看信息.

![效果图](https://github.com/wzk0/photo/blob/main/%E6%88%AA%E5%9B%BE%202023-02-12%2014-30-32.png?raw=true)

## 安装

clone此仓库:

```sh
git clone https://github.com/wzk0/adb-tui
```

随后直接运行`main.py`即可:

```sh
python3 main.py
```

## 使用

按照序号操作即可.
